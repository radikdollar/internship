📌 Описание задачи

Необходимо рассчитать минимальное время спуска ("Незнайка на Луне") с вершины числовой пирамиды (горы) к её основанию. Спуск возможен только на один из двух соседних участков на строке ниже.

Цель программы: Найти минимальную сумму чисел на пути от вершины к основанию.
Вывести саму последовательность чисел (маршрут), которая дает эту минимальную сумму.

🚀 Как работает алгоритм (через Динамическое программирование)

Для решения задачи используется подход "снизу вверх", чтобы гарантированно найти оптимальный (минимальный) маршрут и избежать ошибок локального минимума ("жадного" алгоритма).
Шаги выполнения:
1. Считывание данных: Программа принимает на вход высоту горы N, а затем считывает саму структуру горы в виде двумерного массива (список списков).
2. Поиск минимальной суммы: Создается вспомогательная матрица dp. Начиная с предпоследней строки (снизу вверх), к каждому элементу прибавляется минимальное значение из двух доступных соседних элементов строкой ниже. По итогу цикла в самой верхней ячейке dp[0][0] оказывается минимально возможная сумма всего пути.
3. Восстановление пути: Имея просчитанную матрицу минимальных путей, программа спускается "сверху вниз", на каждом шаге выбирая тот путь, который в матрице dp оказался "дешевле", и записывает элементы исходной горы в итоговый массив.
