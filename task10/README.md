# Задание №10

## Задачи
1. Собрать схему подключения сервопривода.
2. Написать программу для управления сервоприводом через последовательный порт.

## Логика работы
1. При запуске серво устанавливается в 0 градусов, в Serial Monitor появляется
   подсказка "Введите угол от 0 до 180".
2. Пользователь вводит число в Serial Monitor и нажимает Enter.
3. Программа проверяет корректность ввода — принимаются только целые числа
   от 0 до 180. Если введено что-то другое (буква, дробь, число вне диапазона)
   — выводится сообщение об ошибке.
4. Если введён угол, отличный от текущего — серво плавно поворачивается
   (по одному градусу с задержкой 15 мс), в Serial Monitor выводится
   сообщение о начале и конце поворота.

## Как работает
- `Serial.readStringUntil('\n')` — читает всё, что введено до нажатия Enter.
- `input.trim()` — убирает лишние символы (пробелы, \r) по краям строки.
- Проверка `isDigit()` — убеждается, что каждый символ строки является цифрой
  (это и есть "проверка на корректность" из условия задачи).
- Плавный поворот: цикл while увеличивает/уменьшает угол на 1 за раз
  с задержкой `delay(15)` — это даёт плавное движение без рывков.
- Флаг `doorOpen` — нет, здесь флаг `currentAngle` — хранит текущую
  позицию серво, чтобы не крутить его, если угол уже совпадает с заданным.
