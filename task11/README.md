# Задание №11 (Вариант 7)

## Задача
Найти самый большой синий объект на изображении.
Найти центр самого большого синего объекта.
Отметить центр красной точкой.

## Используемые библиотеки
- `opencv-python` (cv2) — обработка изображений
- `numpy` — работа с массивами чисел

## Логика работы

1. **Загружаем изображение** — `cv2.imread()`
2. **Размываем** — `cv2.GaussianBlur()` убирает мелкий шум, чтобы маска
   была чище
3. **Переводим из BGR в HSV** — в HSV гораздо удобнее задавать диапазоны
   цветов. Синий в HSV имеет оттенок (Hue) от 100 до 130.
4. **Создаём маску** — `cv2.inRange()` делает белыми все пиксели, чей
   цвет попадает в диапазон синего, остальное — чёрным
5. **Находим контуры** — `cv2.findContours()` ищет границы белых областей
   на маске
6. **Сортируем по площади** — `sorted(..., key=cv2.contourArea)` — самый
   большой контур окажется первым
7. **Находим центр** — `cv2.boundingRect()` возвращает прямоугольник
   вокруг контура (x, y, w, h); центр = (x + w/2, y + h/2)
8. **Рисуем красную точку** — `cv2.circle()` с цветом (0, 0, 255) — это
   красный в формате BGR

## Почему HSV, а не RGB?
В RGB выбрать "все оттенки синего" сложно — нужно подбирать три числа
одновременно. В HSV синий — это просто один диапазон оттенка (Hue 100-130),
а насыщенность (Saturation) и яркость (Value) можно взять широко.
