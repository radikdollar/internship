Практическое задание №3

1. Постановка задачи
Нужно написать программу для учета снега на улицах города. Программа должна уметь делать две вещи:
Записывать, сколько снега выпало на конкретной улице.
Считать общее количество снега на заданном отрезке улиц.

2. Анализ и выбор метода решения

Тут не нужны сложные алгоритмы, вполне хватит обычного списка. Индекс в списке это номер улицы, а значение это количество снега на ней. Когда приходят новые данные о снеге, мы просто плюсуем их к нужному элементу. А для подсчета суммы на нескольких улицах подряд можно использовать срезы и встроенную функцию sum(). Это пишется легко и отлично подходит для нашей задачи.

3. Алгоритм работы программы

Подготовка: Сначала считываем количество улиц n и количество запросов k. Создаем список streets, который заполняем нулями. Размер делаем n + 1, чтобы индексы четко совпадали с номерами улиц (от 1 до n).

Чтение запросов: Запускаем цикл k раз, чтобы обработать каждый запрос.

Добавление снега: Если первый номер в запросе это 1, значит снег выпал. Берем номер улицы i и количество снега x, а потом просто делаем streets[i] += x.

Подсчет суммы: Если первый номер в запросе это 2, значит надо посчитать сумму. Берем начальную улицу u и конечную r. Делаем срез списка [u:r+1] и считаем его сумму с помощью sum(). Дальше просто выводим результат на экран.
