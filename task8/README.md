# Задание №8
Собрать в Wokwi светодиодную матрицу размером не менее 8x8 из адресных
светодиодов (WS2812B), подключённую к Arduino Uno, и вывести на неё
несколько динамично сменяющихся изображений.

- `diagram.json` — схема для Wokwi (Arduino Uno + матрица `wokwi-led-matrix`
  8x8 с layout "serpentine", подключена к пину 6)
- `sketch.ino` — код скетча

## Как работает
- Используется библиотека **FastLED** для управления адресными
  светодиодами WS2812B.
- Матрица 8x8 в Wokwi задана с атрибутом `"layout": "serpentine"` — это
  значит, что сама деталь уже учитывает змейковую укладку (как на схеме
  в задании: чётные ряды идут слева направо, нечётные — справа налево),
  и в коде индекс светодиода можно считать просто построчно:
  `index = y * 8 + x`. Это и делает функция `XY(x, y)`.
- Каждое изображение задано как массив 8x8 из 0 и 1, где 1 — пиксель
  горит, 0 — погашен.
- Функция `drawImage()` проходит по массиву и зажигает нужные
  светодиоды заданным цветом.
- В `loop()` поочерёдно с задержкой в 1.5 секунды выводятся три
  картинки: улыбающийся смайлик (жёлтый), сердечко (красное) и
  грустный смайлик (синий).
