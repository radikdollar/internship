# Задание №9

## Логика работы
- Зелёный светодиод — двери открываются.
- Красный светодиод — двери закрываются.
- Фоторезистор имитирует приближение/удаление человека от дверей
  (изменяется ползунком в симуляции Wokwi во время запуска).

Алгоритм:
1. По умолчанию горит красный светодиод (дверь закрыта).
2. Микроконтроллер считывает значение фоторезистора с пина A0.
3. Если значение превышает 512 — загорается зелёный светодиод (дверь
   открывается), в Serial Monitor выводится сообщение о событии.
4. Через секунду значение проверяется повторно: если оно всё ещё
   больше 512 — дверь остаётся открытой; если нет — включается красный
   светодиод (дверь закрывается), и в Serial Monitor тоже выводится
   сообщение.

## Подбор номинала резисторов
Использованы резисторы **220 Ом** — стандартный номинал для
ограничения тока через светодиод при питании 5В, обеспечивающий
достаточную яркость без превышения допустимого тока через светодиод
(~20 мА).
